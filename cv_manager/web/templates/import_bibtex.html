How BibTeX Import Works

Publication Types: Automatically categorizes entries based on BibTeX types:

  • @article → Journal Papers
  • @inproceedings, @conference → Conference Papers
  • @misc, @techreport → Preprints
  • @unpublished → Under Review

Data Extraction: Automatically extracts authors, titles, venues, years, pages, DOIs, and other metadata.

Duplicate Detection: Prevents duplicate imports by comparing publication titles.

Sample BibTeX Format

@article{hartford2023example,
  title={An Example Research Paper},
  author={John Doe and Jane Smith},
  journal={Journal of Computer Science},
  volume={42},
  number={3},
  pages={123--145},
  year={2023},
  doi={10.1000/example.doi}
}

@inproceedings{smith2022conference,
  title={Conference Paper Title},
  author={Jane Smith and John Doe},
  booktitle={Proceedings of the International Conference on AI},
  pages={456--467},
  year={2022},
  note={oral presentation -- 25% acceptance rate}
}

@misc{doe2024preprint,
  title={Preprint Paper Title},
  author={John Doe},
  year={2024},
  note={arXiv:2024.12345}
}

The system supports all standard BibTeX entry types and will automatically categorize and format them for your CV.
